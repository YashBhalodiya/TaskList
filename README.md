# ToDo App - Flutter

A simple ToDo app built with Flutter that lets users add, update, and delete tasks. The app features an easy-to-use interface for managing tasks and demonstrates key Flutter concepts like state management, navigation, and user input handling.

## Features

- **Add Tasks**: Users can add new tasks to their to-do list.
- **Update Tasks**: Users can update the details of an existing task.
- **Delete Tasks**: Users can remove tasks from the list.
- **Task List View**: Displays all added tasks with the option to edit or delete them.
